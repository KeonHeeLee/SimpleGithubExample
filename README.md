# SimpleGithubExample

 간단한 깃허브 레포지토리 검색 어플리케이션입니다. 코틀린 및 안드로이드와 관련된 라이브러리를 공부하기 위해 만든 간단한 어플리케이션입니다.
 사용된 라이브러리들은 아래와 같습니다.

## Library
- [x] Android Support Library
- [x] Glide
- [x] Gson
- [x] OkHttp
- [x] Retrofit
- [x] Kotlin Extensions
- [x] Anko
- [x] RxJava
- [x] RxAndroid
- [x] RxBinding
- [x] Lifecycle
- [x] Room
- [x] Dagger2
